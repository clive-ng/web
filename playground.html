Capstone

The undergraduate capstone project is a year-long independent project which allows you top put academic learning into practice.Specifically, this project involves selecting a communication design related problem or issue — and in some cases, a concept — and exploring the optimal means, techniques, and strategies to address it.
Interactive Media Design

I have worked as a freelance developer and designer, handling HTML and CSS coding for responsive web design as well as maintaining legacy Flash websites. I often used Webflow to create the site structure, then exported the code and added custom JavaScript to meet specific client requirements. The most challenging aspect was balancing design with development.
My clients have included the Hong Kong Arts Development Council, Another Mountain Man, MV Production House, Beauty salon, Environmental Association, Geo Park, Sun Hung Kai Properties, Morris, and ING Insurance.
