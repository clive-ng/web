System Workflow

1

Fetch Website

Puppeteer opens the Takachiho event calendar page.

2

Generate Hash

The HTML content is converted into a unique MD5 hash.

3

Compare Changes

The new hash is compared with the previous saved version.

4

Send Notification

If changes exist, an email notification is sent automatically.
